# 什么是 NovaChart？

NovaChart 是一个**为初学者设计**的极简图表库。它从 [Chart.js](https://www.chartjs.org/) 汲取灵感，但把复杂的 API 砍到只剩最常用的部分。

## 一句话定义

> **NovaChart = 配置对象 + 4 种图表 + 3 套主题 + 1 个 `.draw()` 调用**

## 它解决什么问题？

学生写网页作业时，想要展示一组数据（比如考试成绩），但：

- ❌ Chart.js / ECharts 太复杂，光看文档就头大
- ❌ matplotlib 只能在 Python 画，网页展示需要导出图片
- ❌ 自己用 Canvas 画，代码量爆炸

**NovaChart 的答案**：8 个字段、3 行核心代码、搞定 4 种图表。

## 核心设计原则

### 1. 一致性 > 功能多
四种图表的 config 结构 100% 相同。学会 Bar，等于学会 Line/Pie/Table。

### 2. 默认值优先
不填也能跑。`title` / `unit` 等都是可选，库会自动选最合理的默认值。

### 3. 视觉一致性
主题色不只决定图表颜色，还会自动应用到：网格、坐标轴文字、表格表头、Tooltip、斑马纹。**改一处，全局换肤**。

### 4. 单一数据组
三图永远只接受一个 `data: [...]` 数组。没有“数据集”概念，没有多系列叠加。（Table 可传二维自动识别多行）

## 适用场景

✅ **适合**：
- 学生网页作业、数据展示
- 快速原型、教学演示
- 不需要复杂交互的静态可视化
- 后端程序员偶尔前端画图

❌ **不适合**：
- 商业级数据大屏（用 ECharts）
- 复杂的多维度数据分析（用 D3）
- 需要缩放/拖拽的探索式可视化
