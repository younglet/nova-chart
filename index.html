NovaChart极简图表，学一招通四招

为 Python 基础学生打造的轻量图表库 — 4 种图表、3 套主题、零学习成本、表格自动识别一维/二维

快速开始
查看示例
🎯

一致即简洁

Bar / Line / Pie / Table 共用同一套 API。换 type 字段即可切换图表，学会一个会四个。

🎨

三套主题

ocean / sunset / dark（含赛博朋克霓虹风格）。主题色自动应用到图表、网格、表格、Tooltip。

📊

表格也是图表

Table 作为第 4 种图表类型独立存在，不需要 canvas，主题色联动，可直接打印。

🪟

多行表格（重要特性）

`data` 为二维数组时自动渲染多行表格，首列作行名，贴近 pandas DataFrame 风格。不用新增字段，不用记规则。

🪶

极轻量

主库仅 628 行（vs Chart.js 11589 行），18KB，无任何运行时依赖，浏览器原生运行。

🇨🇳

中文友好

标题、标签、单位直接写中文，无需转义。API 命名贴合 matplotlib / pandas 习惯。

🎬

基础动画

fade / slide / none 三种动画，ease-out 缓动，专注核心视觉效果。

一行代码画图 ​

js
new NovaChart('myChart', {
  type: 'bar',
  title: '月考成绩',
  labels: ['语文', '数学', '英语', '物理', '化学'],
  data: [85, 92, 78, 88, 90],
  unit: '分',
  theme: 'ocean'
}).draw();

为什么选择 NovaChart？ ​

相比 Chart.js、ECharts 等"全能型"图表库，NovaChart 做了一件事：砍掉 95% 你用不上的功能，留下最直观的 5%。
