三套主题 ​

NovaChart 内置 3 套主题，只需切换 theme 字段即可全局换肤。

🌊 ocean（海洋）— 默认 ​

明亮清爽，适合大多数场景，护眼首选。

js
new NovaChart('chart', {
  type: 'bar',
  labels: ['语文', '数学', '英语'],
  data: [85, 92, 78],
  theme: 'ocean'   // ← 写或不写都一样，这是默认
}).draw();

配色：

  • 背景：纯白 #FFFFFF
  • 文字：深灰 #2C3E50
  • 网格：浅灰 #ECF0F1
  • 主色板：蓝 / 青 / 橙 / 紫 / 红

🌅 sunset（日落） ​

温暖活力，适合生活类、情感类主题。

js
new NovaChart('chart', {
  type: 'bar',
  labels: ['苹果', '香蕉', '橙子'],
  data: [30, 45, 25],
  theme: 'sunset'
}).draw();

配色：

  • 背景：米白 #FFF9F0
  • 文字：深灰 #333333
  • 网格：浅橙 #FFE8C7
  • 主色板：玫红 / 橙黄 / 鲜黄 / 草绿 / 亮蓝

🌃 dark（赛博朋克） ​

深空黑底 + 霓虹光晕，酷炫护眼（夜间模式）。

js
new NovaChart('chart', {
  type: 'line',
  labels: ['周一', '周二', '周三', '周四', '周五'],
  data: [22, 24, 19, 23, 25],
  theme: 'dark'
}).draw();

配色：

  • 背景：深空黑 #0A0A14
  • 文字：冷光白 #E0E0FF
  • 网格：深紫 #1F1F35
  • 主色板（霓虹）：
    • 🔵 电光青 #00F5FF
    • 🩷 霓虹粉 #FF00E5
    • 🟢 毒液绿 #39FF14
    • 🟠 烈焰橙 #FF6700
    • 🟣 紫光 #BD00FF

额外效果：

  • 柱子、折线、扇形有 shadowBlur 光晕
  • 数值标签加粗
  • 表格边框带辉光

主题如何影响全局？ ​

改 theme 字段，所有视觉元素都会跟着变：

元素受影响
图表柱/线/扇形✅ 用主题色
背景色✅
网格线✅
坐标轴文字✅
数据标签✅
表格表头✅
表格斑马纹✅
表格边框（dark）✅ 霓虹色

颜色循环规则 ​

当数据点超过 5 个，颜色会循环使用：

3 个标签 → 用前 3 色
5 个标签 → 用全部 5 色
7 个标签 → 5 色 + 重复前 2 色

颜色顺序与 theme.palette 数组顺序一致。

实战示例 ​

js
// 一周学习时长折线图（用 dark 主题展示趋势）
new NovaChart('study', {
  type: 'line',
  title: '一周学习时长',
  labels: ['周一', '周二', '周三', '周四', '周五', '周六', '周日'],
  data: [3, 4, 2, 5, 3, 6, 4],
  unit: '小时',
  theme: 'dark'        // 暗色更醒目
}).draw();

自定义主题？（v1 不支持） ​

当前版本不支持自定义主题色板。如果你需要更多主题，请：

  1. 直接修改 nova-chart.js 第 11-37 行的 THEMES 对象
  2. 或在 GitHub 提 Issue 催更
